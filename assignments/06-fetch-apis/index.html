Assignment 6: Exploring Fetch and Public APIs

Requirements

  • Include some basic styling (at least 3 CSS rules).
  • Use fetch to call at least two different APIs.
  • Display the data from both APIs inside your webpage.
  • Include at least one button that fetches new data and updates the page.
  • Your code must:
    • Dynamically update content or styles in the DOM.
    • Extract and display specific fields from the JSON response.
