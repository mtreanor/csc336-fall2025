Assignment 7 – JSON World

Folder: csc336-fall2025/07-json-world/

Overview

In this assignment, you’ll design a small imaginary world in JSON, then write a simple Node.js script that reads and processes it.

Learning Goals

  • Understand how JSON encodes nested data structures
  • Practice writing and validating structured JSON data
  • Use Node.js to read and process data from a file
  • Work with ES Modules and the fs module

Part 1 – Create Your JSON World

Create a file called world.json that describes a small imaginary world using nested JSON. If you are having trouble with understanding JSON, see this: JSON Introduction (W3Schools)

JSON Structure Requirements

  • Your top-level object must have a key called "regions" whose value is an array.
  • Each region must be an object containing:
    • "name" – the name of the region
    • "climate" – a string (e.g. "temperate", "arid")
    • "towns" – an array of town objects
  • Each town must include:
    • "name" – the town’s name
    • "population" – a number
    • "notable_people" – an array of people
  • Each person must include:
    • "name" – their full name
    • "role" – their occupation or title
    • "items" – an array of objects or strings representing their possessions
  • At least one item must be an object with its own properties (for example: an item that has both a "name" and "rarity").
  • Your data should be at least 3 levels deep (e.g., region → town → person → item).

Validation

Before submitting, check that your JSON file is valid:

https://jsonlint.com/

Paste your JSON there and ensure you see “Valid JSON”.

Part 2 – Create a Node Script

In the same folder, create a file called world.js that:

  • Imports the fs module using an ES module import (not require()).
  • Reads the world.json file from disk.
  • Parses the JSON into a JavaScript object.
  • Prints out some interesting subset of the data — for example, all of the "notable_people" names, all town names, or all items with a particular property.
Tip: In order to use import syntax in Node, you must create a file called package.json in the same folder and include this line inside it: 
{
  "type": "module"
}

Then, your import at the top of world.js should look like this:

import * as fs from 'fs'; // or import fs from 'fs';

Run your program in the terminal with:

node world.js
